Entwicklung von Programmen mit grafischer Oberfläche mit MahApps.Metro

MahApps.Metro ist ein Framework, welche Ihre .NET Programme in einem neuen modernen Licht erscheinen lassen. Das Design lehnt sich an das von Microsoft vorgeschlagene Anwendungsdesign für Windows 10 an. Da das Framework eine Open-Source Software ist und ich sehr vom Projekt überzeugt bin, trage ich sehr viel am Code des Frameworks bei.

Was bietet MahApps.Metro?

  • Intuitives UI-Design
  • Konsistente UI-Erfahrung
  • Benutzerdefinierte Auswahl des Anwendungs-Themas (schwarz auf weiß / weiß auf schwarz)
  • Benutzerdefinierte Auswahl des Farbakzentes
